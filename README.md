# 医院挂号系统

## 题目要求

### 题目环境

为提高患者就医效率及医院工作效率，设计并实现网上预约挂号系统。假设医院每天出诊医生有**40人左右**，每天就诊时间为**8:00-12:00**和**14:00-17:00**，预约时间以**半小时**为单位，每半个小时预约人数为**5人左右**，请设计一个预约挂号系统。

### 题目条件

1. 两个角色：管理者、患者
